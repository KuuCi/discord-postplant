Overview

Post Plant automatically tracks Valorant matches for Discord server members who opt-in via Riot Sign On (RSO). When a registered player finishes a match, the bot announces their results to a designated channel.

  • Opt-in only via RSO authentication
  • Announces match results (win/loss, K/D/A, agent, map, score)
  • Groups squad members in the same match into a single announcement
  • Server leaderboards and stat tracking over time

API Endpoints Used

  • RSO OAuth: /authorize for user authentication & opt-in
  • /riot/account/v1/accounts/me: Get authenticated user's PUUID
  • /val/match/v1/matchlists/by-puuid/{puuid}: Get match history for opted-in users
  • /val/match/v1/matches/{matchId}: Get detailed match data

Technical Implementation

  • Platform: Discord.py bot hosted on Railway
  • Game Detection: Discord Rich Presence API
  • Match Data: Riot Valorant API (VAL-MATCH-V1)
  • Authentication: RSO OAuth flow for user opt-in
  • Storage: Persistent JSON storage for registrations

‚úì Policy Compliance

  • Opt-in Required: All users must authenticate via RSO before any data is displayed
  • Privacy Disclaimer: Users are explicitly warned that linking makes their stats visible to server members
  • Post-Match Only: No scouting, no pre-match data, no real-time in-game overlays
  • Approved Use Case: "Discord Bots showing community leaderboards, stats for server members"
  • No Monetization: Completely free, no paid tiers
  • No Gambling: No betting or wagering functionality
